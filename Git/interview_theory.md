# Important Interview Concepts

## Pull vs Fetch vs Rebase vs Merge
git fetch -- Download changes only
git pull -- Fetch + merge
git pull --rebase -- Fetch + rebase
git merge -- Safe merge
git rebase -- Rewrite history

## Reset vs Revert
git reset -- Undo locally (changes history)
git revert -- Safe undo (keeps history)
